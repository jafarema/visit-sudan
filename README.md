# Visit Sudan

> _Sudan, untold._

A cinematic 3D tourism website for Sudan — dark, gold, slow, and emotional. Every section is a scene.

The source for the live site lives in [`visit-sudan/`](./visit-sudan). It is a Vite + React + Three.js project (R3F + Drei + Postprocessing + Lenis + Framer Motion + Tailwind) and is deployed automatically to GitHub Pages on every push to `main` via [`.github/workflows/deploy.yml`](./.github/workflows/deploy.yml).

## Develop locally

```bash
cd visit-sudan
npm install
npm run dev
```

Open http://localhost:5173.

See [`visit-sudan/README.md`](./visit-sudan/README.md) for the full file map, brand tokens, and image-swap workflow.
